# Админ-панель (admin)

Отдельное приложение для управления контентом. Вход только с ролью **admin**.

## Структура проекта

- **client** — сайт для пользователей (`radiant-scroll-works`).
- **server** — бэкенд API (`backend`).
- **admin** — эта панель (добавление/редактирование/удаление тарифов).

## Запуск

1. Убедитесь, что запущен бэкенд (server) на `http://localhost:3001`.
2. Установите зависимости и запустите админку:
   ```bash
   cd admin
   npm install
   npm run dev
   ```
3. Откройте в браузере (по умолчанию порт 5174).
4. Войдите под учётной записью с ролью **admin**. Роль задаётся в базе (см. README в `backend`).

## Функции

- **Вход** — те же учётные данные, что и на клиенте (email + пароль). Токен хранится отдельно (`admin_token`).
- **Проверка роли** — после входа проверяется `user.role === 'admin'`. Если роль `client` или `manager`, показывается «Доступ запрещён».
- **Тарифы (Pricing)** — список, добавление, редактирование, удаление. Доступно только админу; API на бэкенде тоже проверяет роль.

Переменная окружения `VITE_API_URL` (например в `.env`) — URL бэкенда, по умолчанию `http://localhost:3001`.
